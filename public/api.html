FOCO Venezuela — API de datos abiertos

Datos públicos de necesidades y recursos tras el doblete sísmico. Solo lectura, sin datos personales.
No es un servicio de emergencia. FOCO coordina información de ayuda ciudadana; no garantiza respuesta ni rescate. Ante peligro inmediato, contacta a las autoridades.

Estos datos son abiertos para que medios, ONGs, otros mapas e investigadores los usen. Uso libre con atribución a focovenezuela.org. Las respuestas se sirven sobre HTTPS, con CORS abierto y cacheadas ~5 minutos.

ÍndiceJSON/api/
Privacidad. La API nunca expone contacto ni coordenadas exactas (viven aparte, solo visibles para coordinadores verificados). La ubicación es a nivel de sector (~1 km), salvo sitios públicos como edificios dañados, donde precision es "exacta". El texto usa un resumen saneado de datos personales cuando está disponible.

Campos — necesidad

creada_en · actualizada_enMarcas de tiempo ISO-8601 (UTC).

Ejemplo

fetch('https://focovenezuela.org/api/necesidades.json')
  .then(r => r.json())
  .then(d => {
    console.log(d.total, 'necesidades');
    for (const n of d.necesidades) {
      console.log(n.categoria, n.urgencia, n.sector, n.lat, n.lng);
    }
  });

Notas de uso

Solo lectura (GET). Las respuestas se cachean ~5 minutos: no consultes con una frecuencia mayor. Las necesidades duplicadas (depuradas por el curador) no se incluyen.
