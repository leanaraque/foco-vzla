FOCO Venezuela — Open data API

Datos públicos de necesidades y recursos tras el doblete sísmico. Solo lectura, sin datos personales.
Public data on needs and resources after the earthquake doublet. Read-only, no personal data.
No es un servicio de emergencia. FOCO coordina información de ayuda ciudadana; no garantiza respuesta ni rescate. Ante peligro inmediato, contacta a las autoridades.
Not an emergency service. FOCO coordinates citizen-aid information; it does not guarantee a response or rescue. In case of immediate danger, contact the authorities.

Estos datos son abiertos para que medios, ONGs, otros mapas e investigadores los usen. Uso libre con atribución a focovenezuela.org. Las respuestas se sirven sobre HTTPS, con CORS abierto y cacheadas ~5 minutos.

This data is open for use by media, NGOs, other maps and researchers. Free to use with attribution to focovenezuela.org. Responses are served over HTTPS, with open CORS and cached for ~5 minutes.

Privacy. The API never exposes contact info or exact coordinates (these live apart, visible only to verified coordinators). Location is at sector level (~1 km), except for public sites such as damaged buildings, where precision is "exacta". The text uses a PII-sanitized summary when available; descripcion_en carries the English version when present.

Ejemplo

Example

fetch('https://focovenezuela.org/api/necesidades.json')
  .then(r => r.json())
  .then(d => {
    console.log(d.total, 'necesidades');
    for (const n of d.necesidades) {
      console.log(n.categoria, n.urgencia, n.sector, n.lat, n.lng);
    }
  });

Notas de uso

Usage notes

Solo lectura (GET). Las respuestas se cachean ~5 minutos: no consultes con una frecuencia mayor. Las necesidades duplicadas (depuradas por el curador) no se incluyen.

Read-only (GET). Responses are cached for ~5 minutes: do not poll more often. Duplicate needs (deduplicated by the curator) are not included.
